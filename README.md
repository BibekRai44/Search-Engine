## Run Locally

Clone the project

```bash
  git clone https://github.com/Bibekkun/Search-Engine
```
Go to the project directory

```bash
   cd Search-Engine
```
```bash
   start index.html
```

All Done!

**Deployment is for frontend only. Real working is locally**
